Instrucciones para resolver el control

El programa de prueba para la pregunta 1 está en el directorio P1.
Programe la solución correcta en el archivo P1/buf.c.  Ya viene programada
la versión incorrecta, que sirve además como plantilla para su solución
correcta.

El programa de prueba para la pregunta 2 está en el directorio P2.
Cree el archivo P2/reservar.c y programé ahí su solución.  La plantilla
para la solución es P2/reservar.c.plantilla.

El programa de prueba para la pregunta 3 está en el directorio P3.
Cree el archivo npsystem/nsrc/nReservar.c y programe ahí su solución.
La plantilla para la solución es npsystem/nsrc/nReservar.c.plantilla.

Ejecute make en los directorios P1, P2 y P3 para recibir instrucciones
adicionales.

Para eliminar todos los archivos que se pueden generar:

make clean
